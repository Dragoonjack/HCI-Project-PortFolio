Campus Cafeteria Rating System

Bridging the gap between unstructured human sentiment and structured quantitative data.

The Problem

Traditional 5-star rating systems are inherently flawed. A student might leave a review stating, "The food was quite good, but the service was incredibly slow and the portions were slightly small." Forcing this nuanced, unstructured data into a rigid numerical score often leads to skewed campus analytics and unhelpful metrics for cafeteria management.

System Architecture & NLP

To solve this, I engineered a two-part intelligent system. First, the application utilizes Natural Language Processing (NLP) to parse written student reviews. The NLP module tokenizes the text and performs sentiment analysis to extract "crisp" numerical values based on three distinct parameters: Food Quality, Service Speed, and Value for Money.

The Fuzzy Logic Inference Engine

Because human language includes modifiers (like "slightly" or "incredibly"), standard Boolean logic (True/False) is insufficient. I passed the NLP outputs into a Fuzzy Inference System. By defining specific membership functions (Poor, Average, Excellent) and establishing a robust matrix of IF-THEN rules, the system intelligently defuzzifies the data, producing a highly accurate, bias-reduced final rating out of 100%.

System Features

  • Automated Sentiment Extraction
  • Custom Fuzzy Rule Matrix
  • Bias Reduction Algorithms
  • Scalable Data Processing
